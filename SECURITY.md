# Security Policy

## Reporting Vulnerabilities

If you discover a security vulnerability, please report it privately to **founders@compile.now**. Do not open a public issue.

We will acknowledge receipt within 48 hours and provide an initial assessment within 7 days.

## Scope

This policy covers:
- The compile.now web platform
- The open source codebase
- The API and backend services
- The experiment pipeline and data handling

## Research Data

- All connectome data used by Compile is publicly available (FlyWire, MICrONS)
- No private or proprietary biological data is stored or processed
- Circuit designs and growth programs are computational outputs, not biological materials
- All results are from simulation — no biological tissue is involved

## Responsible Disclosure

We ask that you:
- Allow reasonable time for us to address the issue before public disclosure
- Avoid accessing or modifying other users' data
- Act in good faith to avoid disruption to the platform
