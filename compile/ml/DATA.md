# Data Setup

This document explains exactly which data files are needed, where to get them, and where to place them.

## Quick start

```bash
mkdir -p data
# Place all downloaded files in ./data/
# OR set the environment variable:
export COMPILE_DATA_DIR=/path/to/your/data
```

## Required files (FlyWire connectome)

All fly experiments require these three files from the FlyWire v783 release:

| File | Size | Source | Used by |
|------|------|--------|---------|
| `2025_Connectivity_783.parquet` | ~1.5 GB | [FlyWire](https://flywire.ai) > Downloads > Connectivity | All fly experiments |
| `2025_Completeness_783.csv` | ~15 MB | [FlyWire](https://flywire.ai) > Downloads > Completeness | All fly experiments |
| `flywire_annotations.tsv` | ~50 MB | [FlyWire](https://flywire.ai) > Downloads > Annotations | Gene-guided, controls, growth |

### Module labels

| File | Size | Source | Used by |
|------|------|--------|---------|
| `module_labels_v2.npy` | ~1 MB | Generated by MiniBatchKMeans on 4 neuron features (see below) | Edge sweep, evolution, interference matrix |

Module labels are NOT a standard FlyWire download. They are computed by **MiniBatchKMeans** (n_clusters=50, random_state=42, batch_size=10000) on a 4-feature representation of each neuron (normalized index, log in-degree, log out-degree, balance ratio). To regenerate:

```bash
python scripts/compute_module_labels.py
```

Or set: `export COMPILE_MODULE_LABELS=/path/to/module_labels_v2.npy`

## Optional files (MICrONS mouse cortex)

Cross-species experiments (`cross_species/`) require MICrONS data:

| File | Source | Used by |
|------|--------|---------|
| MICrONS node/edge data | [CAVEclient](https://github.com/CAVEconnectome/CAVEclient) API | `exp5_microns.py`, `mouse_full_pipeline.py` |

MICrONS data is loaded via `caveclient` (requires authentication) or from cached pickle files. If neither is available, `exp5_microns.py` falls back to synthetic data for development.

```bash
pip install caveclient
export MICRONS_DATA_DIR=data/microns
```

## File placement

The data loader (`compile.data`) searches these locations in order:

1. `$COMPILE_DATA_DIR` (environment variable — recommended)
2. `./data/` (relative to working directory)
3. `./latent/ml/data/` (if running from repo root)

For module labels:
1. `$COMPILE_MODULE_LABELS` (environment variable)
2. `./data/module_labels_v2.npy`

For MICrONS:
1. `$MICRONS_DATA_DIR` (environment variable)
2. `./data/microns/`

## Error messages

If data is missing, you'll see a clear error like:

```
FileNotFoundError: Cannot find connectome data directory.
Set COMPILE_DATA_DIR environment variable or place data files in ./data/.
Required files:
  - 2025_Connectivity_783.parquet
  - 2025_Completeness_783.csv
  - flywire_annotations.tsv
Download from https://flywire.ai
```

Tests use synthetic data and do not require any downloads.

## Data not included in repository

The `.gitignore` excludes `*.parquet`, `*.npy`, and `data/` to avoid committing large binary files. All data must be downloaded separately.
